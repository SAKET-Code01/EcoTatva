Global UI Shell (Applies to All Screens)

🗂️ Left Sidebar Navigation

  • Fixed vertical sidebar with EcoTatva logo + leaf icon at top
  • Icon + label nav items: Dashboard, Tasks, Learn, Games, Leaderboard, Community, Events, Articles, Badges, Profile, EcoBot, Settings
  • Active state: filled green background pill with white text + icon
  • Inactive state: muted icon + text, hover shows light green tint
  • Bottom of sidebar: user avatar thumbnail + display name + rank badge

🔝 Top Header Bar

  • Stretches full content-area width across every page
  • Left: page title (e.g. "Dashboard", "Community")
  • Centre: Search bar with placeholder "Search EcoTatva…"
  • Right pills (left → right): ⚡ XP count (green pill) · 🔥 Streak count (amber pill) · 🔔 Notification bell (with badge dot) · 👤 User avatar (circle, opens profile menu)

Screen-by-Screen User Journeys

🆕 New User Onboarding

  • Lands on full-screen splash with EcoTatva logo + leaf animation
  • Google / Email sign-in via Firebase Auth
  • Picks mode: Kids (6–12) or Teen / Adult (large illustrated cards)
  • Sets display name + picks avatar from preset eco-themed set
  • Short 3-slide animated intro explaining XP, streaks & tasks
  • Lands on Dashboard — sidebar visible, top header active

🏠 Dashboard (Home)

  • 4 metric cards in a row: Total XP · Current Streak · CO₂ Saved · Tasks Completed
  • Each card has a large colored icon, number, label, and a small trend arrow
  • Below: "Today's Tasks" section — 3 task cards with category icon, title, XP reward chip, and "Complete" button
  • Sidebar active state highlights "Dashboard"
  • Top header shows live XP and streak values

✅ Tasks Page

  • Header: "Your Eco Tasks" with date badge
  • 3 AI-personalized task cards — each shows: emoji icon, task title, short description, difficulty chip (Easy/Medium/Hard), XP reward, and a "Mark Complete" button
  • Completed tasks show green checkmark overlay + strikethrough
  • Text proof submission inline in card (expandable textarea)
  • Gemini validation feedback shown as a green banner after submit

📚 Learn Page

  • Featured lesson banner at top (large card with gradient + topic image)
  • Category grid below: Waste Management · Water Conservation · Energy Saving (each as a colored card with icon and lesson count badge)
  • Tapping a category opens lesson list; tapping a lesson opens full content + 5-question Gemini quiz
  • Quiz result screen shows score ring, correct/wrong breakdown, XP earned

🏆 Leaderboard Page

  • Top section: Podium display — 2nd place (left, silver), 1st place (centre, elevated gold), 3rd place (right, bronze); each shows avatar, name, and XP total
  • Below podium: scrollable ranked table with rank number, avatar, name, XP, streak, rank badge
  • Toggle tabs: Global / Local (city-level)
  • Current user row highlighted in light green

🤝 Community Feed

  • Feed of user achievement posts with avatar, display name, timestamp
  • Posts can include: task completion, badge unlock, game score, or free-text eco update
  • Optional image attachment visible in feed card
  • Like ❤️ count, 💬 comment count, hashtag chips (#WasteWarrior, #EcoTatva)
  • Floating "+" button → Create Post modal (text + optional image + hashtag picker)

📅 Events Page

  • Grid of event cards — each shows: cover image, event title, date + time, location tag, organiser name
  • Green "Join Event" button on each card
  • Filter tabs: Upcoming · Nearby · Online
  • Joined events show a "Joined ✓" badge replacing the button

📰 Articles & Updates

  • Curated eco-news and sustainability articles with thumbnail, title, source tag, and read-time chip
  • Category filter tabs: All · Waste · Water · Energy · Climate
  • Tapping an article opens full-read view inside app
  • Bookmark icon saves articles to profile reading list

🏅 Badges Page

  • Grid of badge cards — earned badges show full colour + unlock date; locked badges shown as desaturated with a lock icon overlay
  • Each badge card has a circular progress ring showing progress toward unlock
  • Categories: Streak Badges · Task Badges · Quiz Badges · Game Badges · Special
  • Tapping a badge shows modal: name, description, criteria, how close user is to earning it

👤 Profile Page

  • Top section: avatar (large), display name, rank badge, location tag, "Edit Profile" button
  • Stats row: Total XP · Longest Streak · Tasks Done · Badges Earned
  • Recent Activity timeline: last 5 completed tasks / earned badges with timestamps
  • Achievements section: pinned top badges shown as icon strip
  • Account mode indicator (Kids / Adult) with toggle option

🤖 EcoBot Chatbot

  • Full chat interface: message bubbles (user right, EcoBot left with leaf avatar)
  • EcoBot powered by Gemini — answers eco questions, gives task hints, explains quiz answers
  • Quick-reply chips at bottom: "What's my streak?", "Give me a task hint", "Today's eco tip"
  • Text input bar + send button at bottom
  • Chat history preserved per session in Firestore

⚙️ Settings Page

  • Sections: Account · Notifications · Appearance · Privacy · About
  • Account: edit name, avatar, age group, location, Kids/Adult mode toggle
  • Notifications: toggle push alerts for daily tasks, streak reminders, community likes
  • Appearance: light / dark mode toggle (follows system by default)
  • Privacy: view privacy policy, delete account + all data button (with confirmation)

🎮 Mini Game

  • Goes to Games tab via sidebar
  • Plays Smart Waste Sorting — items fall from top of screen
  • Drags items into correct coloured bin (Dry / Wet / Hazardous / E-Waste)
  • 60-second countdown timer adds urgency; speed increases with level
  • 3 strikes = game over screen with score, XP earned, personal best indicator
  • "Share to Community" button auto-creates a feed post with score

→ Writes tasks to Firestore under users/{uid}/tasks/{date}
→ Sends FCM push notification "Your eco-tasks are ready!"

Database Schema (Firestore Collections)

Firestore is a NoSQL document database. Each top-level item below is a Collection. Sub-items are Documents with the listed fields.
👤 users ›
{
  uid: string,               // Firebase UID (also the document ID)
  email: string,
  displayName: string,
  photoURL: string,
  mode: "kids" | "adult",    // UI mode
  ageGroup: "6-8"|"9-12"|"13-17"|"18+",
  location: { city, state, country },
  totalPoints: number,       // running total
  weeklyPoints: number,      // resets every Monday
  currentStreak: number,     // consecutive days with ≥1 task done
  longestStreak: number,
  rank: string,              // "Seed"|"Sprout"|"Sapling"|"Tree"|"Forest Guardian"
  badges: [{ id, name, earnedAt }],
  preferredTopics: ["waste","water","energy"],
  createdAt: Timestamp,
  lastActiveAt: Timestamp,
  fcmToken: string,          // for push notifications
  isKidAccount: boolean,
  parentEmail: string        // optional, for kids accounts
}
📅 tasks/{uid}/daily/{YYYY-MM-DD} ›
{
  date: string,              // "2026-04-25"
  tasks: [
    {
      taskId: string,        // uuid
      title: string,         // "Save one bottle of water today"
      description: string,
      category: "waste"|"water"|"energy",
      difficulty: "easy"|"medium"|"hard",
      pointsReward: number,  // 10, 20, 30
      isCompleted: boolean,
      completedAt: Timestamp | null,
      proofType: "text"|"photo"|"none",
      proofUrl: string | null,
      geminiValidated: boolean,
      geminiScore: number,   // 0-100, confidence of validity

Complete API Design

Base URL: https://ecotatva-api.onrender.com/api/v1 — All endpoints require Authorization: Bearer {firebase_id_token} header except /auth/verify.

MethodEndpointPurposeResponse Example
POST/auth/verifyVerify Firebase token, create user doc if new{ uid, isNew, profile }
GET/users/meGet own profile + stats{ uid, points, streak, badges, rank }
PUT/users/meUpdate profile (name, mode, location){ success: true }
GET/tasks/todayGet today's 3 tasks for auth user{ date, tasks: [Task] }
POST/tasks/{taskId}/completeSubmit task completion with optional proof{ pointsEarned, feedback, newBadges }
GET/lessonsList available lessons (3 topics){ lessons: [LessonSummary] }
GET/lessons/{id}Full lesson content + quiz questions{ lesson: Lesson }
POST/lessons/{id}/quizSubmit quiz answers, get score + feedback{ score, correctCount, pointsEarned }
GET/leaderboard/globalTop 50 users globally (weekly points){ entries: [LeaderEntry], myRank: number }
GET/leaderboard/localTop 50 in user's city{ entries: [...], myRank: number }
GET/impact/mePersonal eco-impact summary{ co2Saved, waterSaved, treesEquiv, tasksDone }
POST/game/scoreSubmit mini game score{ pointsEarned, personalBest: bool }
GET/badgesAll available badges + user's earned ones{ badges: [Badge] }
GET/ai/task-hint/{taskId}Gemini hint for how to complete a task{ hint: string }

# services/reward_engine.py

RANK_THRESHOLDS = [
    (0, "Seed"), (200, "Sprout"), (500, "Sapling"),
    (1000, "Tree"), (2500, "Forest Guardian")
]

BADGE_CRITERIA = {
    "first_task": {"type": "task_count", "value": 1},
    "streak_3": {"type": "streak", "value": 3},
    "streak_7": {"type": "streak", "value": 7},
    "waste_warrior": {"type": "category_tasks", "category": "waste", "value": 5},
    "water_saver": {"type": "category_tasks", "category": "water", "value": 5},
    "energy_hero": {"type": "category_tasks", "category": "energy", "value": 5},
    "quiz_master": {"type": "perfect_quizzes", "value": 3},
    "daily_champion": {"type": "all_tasks_days", "value": 7},
}

async def award_points_and_badges(uid: str, points: int, context: dict) -> dict:
    user = await get_user(uid)
    user.totalPoints += points
    user.weeklyPoints += points
    
    # Streak update
    today = date.today()
    if user.lastActiveAt.date() == today - timedelta(days=1):
        user.currentStreak += 1
    elif user.lastActiveAt.date() != today:
        user.currentStreak = 1
    
    if user.currentStreak > user.longestStreak:
        user.longestStreak = user.currentStreak
    
    # Streak milestones
    if user.currentStreak in [7, 30]:
        bonus = 50 if user.currentStreak == 7 else 150
        user.totalPoints += bonus
    
    # Rank check
    new_rank = calculate_rank(user.totalPoints)
    
    # Badge check
    new_badges = await check_badges(uid, user, context)
    
    await update_user(uid, user)
    await update_leaderboard(uid, user)
    
    return {"pointsAdded": points, "newRank": new_rank, "newBadges": new_badges}

}

// Item pool (expand with Gemini later)

  { id: "battery", name: "Battery", emoji: "🔋", bin: "hazardous", points: 15 },
  // ... 20 more items
]

// Core game loop
useEffect(() => {
  if (!gameState.isRunning) return;
  const interval = setInterval(() => {
    spawnNewItem();           // add item to queue
    setTimeLeft(t => t - 1); // countdown
    if (timeLeft <= 0) endGame();
  }, 1000);
  return () => clearInterval(interval);
}, [gameState.isRunning]);

// Drag and drop with touch support
const handleDrop = (binId) => {
  if (binId === currentItem.bin) {
    setScore(s => s + currentItem.points * level);
    showSuccess();
  } else {
    setStrikes(s => s + 1);
    showError("That goes in " + BIN_LABELS[currentItem.bin]);
    if (strikes >= 2) endGame();
  }
};
